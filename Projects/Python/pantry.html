See below for User Guide



Pantry:
In order to add to the pantry, type "add pantry ingredient amount ingredient amount ..."
In order to see current pantry, type "list pantry"
In order to remove ingredients from the pantry, type "remove pantry ingredient ingredient ..."

Recipes:
In order to create a new recipe, type "new recipe recipe_name ingredient amount ingredient amount ..."
In order to see all current recipes, type "list recipes"
In order to remove a recipe, type "remove recipes recipe recipe ..."

Shopping List:
In order to add to the shopping list, type "add shopping list ingredient amount ingredient amount ..."
In order to see current shopping list, type "list shopping list"
In order to remove ingredients from the shopping list, type "remove shoppping list ingredient ingredient ..."

Individually Feasible Recipes:
In order to see what recipes you can make individually, type "individually feasible"

Simultaneously Feasible Recipes:
In order to see what recipes you can make simultaneously, type "simultaneously feasible"

Needed Ingredients:
In order to see what ingredients you need (not already in the pantry) to make simultaneous recipes, type "needed ingredients recipe recipe ..."
